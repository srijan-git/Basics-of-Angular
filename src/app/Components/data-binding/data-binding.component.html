Data Binding

Data binding is the way to communicate between your Typescript code of your component and your HTML view.

Types of Data Binding

1.Interpolation

2.Property Binding

3.Event Bindng

4.Two way Binding

1. What is {{ bindingTechnique }} ?

When ever we want to trasfer data from Typescript file to the HTML file we can use Interpolation.It is a One way data-binding technique

2. What is Property Binding ?

The use of Property Binding is similar like interppolation but it bindes the property of HTML attributes.

2. What is Event Binding ?

Event binding is used to handle the events raised by the user action like button click.

2. What is Two way binding ?

Two-way data binding in Angular will help users to exchange data from the view to component and then from component to view.
